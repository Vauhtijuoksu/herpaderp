# HEartRatePythonAsyncDeviceReaderProgram

This branch is dedicated for running herpaderp in external machine and serving heart rates to web

## How to install:

Create venv
```
python -m venv ./venv 
```
Activate venv
```
source venv/bin/activate
```
Install deps
```
pip install pip3-deps 
```
## How to run:
```
./launch.sh
```
